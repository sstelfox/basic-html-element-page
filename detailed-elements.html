HTML5 Detailed Elements Page

This is a completely unstyled, validatable HTML5 page that has the majority of the elements defined by the W3C. The purpose is two-fold, providing designers with a quick page to look at how their stylesheets affect all the various elements throughout a page, and at the same time provide a reference of valid attributes for each tag, where it belongs, and what it's intended purpose is.

Of course if you have any design specific elements you will need to extend this page with your own markup to truly make use of it.

Global Attributes

These attributes apply to all HTML5 elements.

Attribute Values Description Browser Support
accesskey Alphanumeric character Specifies a shortcut key to activate/focus an element using Alt + key or Shift + Alt + Key. All major browsers except for Opera.
class Specifies one or more class names for an element. To specify multiple classes, separate the class names with a space. Must begin with an ascii letter, can be followed by: letters, digits, hyphens, and underscores. All values are case-insensitive. Specifies one or more classnames for an element (refers to a class in a style sheet). Supported by all major browsers.
contenteditable true, false, inherit Specifies whether the content of an element is editable or not. Supported by all major browsers.
contextmenu The ID of a "menu" tag on the same page. Specifies a context menu for an element. The context menu appears when a user right-clicks on the element. Not supported by any browsers.
dir ltr, rtl, auto Specifies the text direction for the content in an element (left to right, right to left, auto). Supported by all major browsers.
draggable true, false, auto Specifies whether an element is draggable or not. Currently supported in Firefox, Chrome, and Safari.
dropzone copy, move, link Specifies whether the dragged data is copied, moved, or linked, when dropped. Not supported by any browsers.
hidden N/A Specifies that an element is not yet, or is no longer, relevant. Currently supported in all major browsers, except Internet Explorer.
id Must be unique. Must contain at least one character. Must not contain any space characters. Case-insensitive. Specifies a unique id for an element. Supported by all major browsers.
lang Valid language code (two digits). Specifies the language of the element's content. Supported by all major browsers.
spellcheck true, false Specifies whether the element is to have its spelling and grammar checked or not. Currently supported in Opera, Chrome, and Safari.
style One or more CSS properties and values separated by semicolons. Specifies an inline CSS style for an element. Supported by all major browsers.
tabindex number (should be unique and start at 1) Specifies the tabbing order of an element. Supported by all major browsers, except Safari.
title text Provide a tooltip text for an element Supported by all major browsers.
