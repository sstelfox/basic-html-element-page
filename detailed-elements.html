HTML5 Detailed Elements Page

This is a completely unstyled, validatable HTML5 page that has the majority of the elements defined by the W3C. The purpose is two-fold, providing designers with a quick page to look at how their stylesheets affect all the various elements throughout a page, and at the same time provide a reference of valid attributes for each tag, where it belongs, and what it's intended purpose is.

Of course if you have any design specific elements you will need to extend this page with your own markup to truly make use of it.

Attributes Reference

Attribute Description Valid Values Supported Tags Browser Support
accesskey Specifies a shortcut key to activate/focus an element using Alt + key or Shift + Alt + Key. A single valid value. All All major browsers except for Opera
class Specifies one or more classnames for an element (refers to a class in a style sheet). Specifies one or more class names for an element. To specify multiple classes, separate the class names with a space. Must begin with an ascii letter, can be followed by: letters, digits, hyphens, and underscores. All values are case-insensitive. All Supported by all major browsers
contenteditable Specifies whether the content of an element is editable or not. true, false, inherit All Supported by all major browsers
contextmenu Specifies a context menu for an element. The context menu appears when a user right-clicks on the element. The ID of a "menu" tag on the same page. All Not supported by any major browser
dir Specifies the text direction for the content in an element (left to right, right to left, auto). ltr, rtl, auto All Supported by all major browsers
draggable Specifies whether an element is draggable or not. true, false, auto All Currently supported in Firefox, Chrome, and Safari
dropzone Specifies whether the dragged data is copied, moved, or linked, when dropped. copy, move, link All Not supported by any browsers
hidden Specifies that an element is not yet, or is no longer, relevant. N/A All Currently supported in all major browsers, except Internet Explorer
id Specifies a unique id for an element. Must be unique. Must contain at least one character. Must not contain any space characters. Case-insensitive. All Supported by all major browsers
lang Specifies the language of the element's content. Valid language code (two digits). All Supported by all major browsers
spellcheck Specifies whether the element is to have its spelling and grammar checked or not. true, false All Currently supported in Opera, Chrome, and Safari
style Specifies an inline CSS style for an element. One or more CSS properties and values separated by semicolons. All Supported by all major browsers
tabindex number (should be unique and start at 1) Specifies the tabbing order of an element. All Supported by all major browsers, except Safari
title Provide a tooltip text for an element text All Supported by all major browsers
